??? note "Build Info"
    * Adhoc Version: `3`
    * Config: `prius`
    * Supports up to instruction: `VARIABLE_PUSH`
    * BuildNumber: `62`
    * Game Code: `PAPX-90512`

    * ISO Size: `1.41 GB`
    * ISO CRC: `E091C9AA`
